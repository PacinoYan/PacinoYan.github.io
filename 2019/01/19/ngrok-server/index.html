搭建ngrok服务器

发表于 2019-01-19 | 分类于 ngrok

ngrok简介

本文使用的是阿里云的ECS云服务器，操作系统为Ubuntu16.04LTS，没有云服务器的同学可以登录阿里云控制台进行学生身份认证，通过以后可以以9.9/月的价格购买阿里云的学生机（云翼计划），非常划算。链接如下https://promotion.aliyun.com/ntms/act/campus2018.html。
